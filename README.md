# nwt-project
Installer les prérequis : 
```
cd front
yarn

cd back
yarn
```

Lancer l'application : 
```
cd front
ng serve

cd back
npm run start
```

Installer la base de données :
```
créer un bdd mongoDB appelé "nwt"

lors du premier lancement du back, 
la base de données sera automatiquement créée
```
